Form Lab - Learning Form Elements

Action

This attribute specifies the server URL that will receive the form data from submission. It determines where to send the form data when the submit button is pressed.

Form Submission

Autocomplete

Autocomplete helps fill out forms based on previous entries. Turning it off prevents the browser from completing fields automatically and even enhances privacy.

Method

The method attribute controls how data is sent to the server. Using "post" hides the data from the URL, which even enhances security compared to using "get".

Submission Method

Enctype

This attribute specifies how the browser encodes the form data when sending it to the server. "multipart+form-data" is necessary for uploading files as it handles file data correctly.

File Upload

ID

The ID attribute provides a unique identifier for form elements, which is useful for CSS styling. It ensures each form element can be uniquely accessed without worry of not fidning it.
